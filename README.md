# NPM Package Converter

NPM Package Converter is a simple Electron-based desktop application that helps you save the README content of an npm package page as a Markdown file.

## Overview

This project opens the npm website inside an Electron window and adds a floating button to the page. When you click the button, the current package URL is captured and the package README is fetched from the npm registry and saved into the output folder.

## Features

- Opens npm package pages in an Electron window
- Adds a floating save button to the page
- Fetches package metadata from the npm registry
- Saves the package README as a Markdown file
- Opens the output folder automatically after saving

## How It Works

1. The Electron app launches a browser window pointing to npmjs.com.
2. A custom button is injected into the page.
3. Clicking the button sends the current page URL to the main process.
4. The app extracts the package name from the URL.
5. The package README is fetched and written to the output directory.

## Requirements

- Node.js
- npm
- Electron

## Installation

Install dependencies:

```bash
npm install
```

## Running the App

Start the application:

```bash
npm start
```

Build the application:

```bash
npm build
```

## Output

Saved Markdown files are written to the output folder in the project directory.

## Project Structure

- index.js - Main Electron process
- preload.js - Script injected into the browser window
- package.json - Project metadata and dependencies
- output/ - Folder where generated Markdown files are stored

## Notes

- The application currently expects npm package URLs in the form https://www.npmjs.com/package/<package-name>.
- Saving uses the package README content directly, so the output file may vary depending on the package metadata.
- On Windows, the app opens the output folder using the default file explorer.
